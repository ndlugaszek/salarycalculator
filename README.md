# **Kalkulator wynagrodzeń**
```
# W celu uruchomienia programu, należy wykonać poniższe kroki:
1) pobrać archiwum z programem
2) rozpakować archiwum
3) otworzyć folder salarycalculator-main/salacycalculator-main
3) będąc w powyższym folderze, należy wywołać poniższe komendy:

# instalacja bibliotek:
$ pip install -r requirements.txt

# uruchomienie programu:
$ python main.py [-L] N [N ...]

# przykład:
$ python main.py -L 10000 9500 7800
```
